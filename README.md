# QFNU-JWC-Tracker

曲阜师范大学教务处公告监控

每 5 分钟检查一次教务处公告，有新公告时提取标题，摘要，链接，推送至群聊

qfnujwc-on 开启教务处公告监控

qfnujwc-off 关闭教务处公告监控

qfnuzcc-on 开启资产处公告监控

qfnuzcc-off 关闭资产处公告监控
